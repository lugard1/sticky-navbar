# sticky-navbar

## Learning Objectives
- Understand how to replicate a Sticky Navbar application which is used to calculate the window's scroll position that enables the navbar to switch background colors at certain height of the window scroll, and it supports excellent user experience, and has beautiful user interface.

## Description of the Project
For the first milestone in the process of creating sticky navbar, you will:
- set up a new repository and prepare it for development using best practices.
